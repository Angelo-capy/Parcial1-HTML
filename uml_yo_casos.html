Descripción del Diagrama de Clases

Sistema de Venta de Beats

  • Administrator: Gestiona el sistema completo con funciones de administración
  • Organization: Organizador con acceso limitado al inventario
  • DBVenta: Base de datos para registrar y consultar ventas de licencias
  • DBInventario: Base de datos para gestionar el stock de beats y licencias
  • Comprador/Cliente: Usuario final que compra licencias de beats
  • Pedido: Representa las transacciones de compra de licencias
